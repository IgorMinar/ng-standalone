Angular Standalone Component Demo

This is a simple demo project, that shows what a standalone @Component, @Directive, and @Pipe in Angular could look like in the future.

The demo is built by wrapping an existing @Component decorator and generating a hidden (virtual) NgModule. The application, including the standalone component, is then compiled using the JIT compiler.
